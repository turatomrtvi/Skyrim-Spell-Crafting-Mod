## Features
- Create your own spells
- Combine effects for unique spells
- Spell learning and leveling system

## Code example
Python
```python
class Spell:
def __init__(self, name, effects):
self.name = name
self.effects = effects

def cast(self):
print(f"Casting {self.name} with effects: {', '.join(self.effects)}")
```
